## 学習内容

### 1. Node.js と npm とは？

**Node.js**: JavaScript をブラウザ外（サーバーサイド）で実行できる環境

**npm (Node Package Manager)**: JavaScript のパッケージ管理ツール

### 2. Vite とは？

Vite（ヴィート）は、高速な開発環境を提供するビルドツールです。

**メリット**:
- 超高速な起動（従来のツールより10倍以上速い）
- ホットモジュールリプレースメント（HMR）が速い
- シンプルな設定

**Create React App との比較**:
- Create React App: 古いツール、起動が遅い
- Vite: モダン、高速、推奨
