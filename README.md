# 100日間 React マスター学習プログラム

## 📚 概要

このリポジトリは、**100日間で React をマスターする**ための包括的な学習教材です。
初心者から実務レベルまで、段階的にスキルアップできるように設計されています。

## 🎯 学習目標

100日後には以下のスキルが身につきます：

- ✅ React 19 / Next.js 15 でのモダン開発
- ✅ TypeScript による型安全な開発
- ✅ TanStack Query でのサーバー状態管理
- ✅ Tailwind CSS + shadcn/ui でのモダン UI
- ✅ テスト駆動開発
- ✅ Vercel での本番デプロイ

---

## 📖 使い方

### 1. クイックスタート

まずは `QUICKSTART.md` を読んで、学習環境をセットアップしましょう。

### 2. 学習計画を確認

`100-day-react-plan.md` で全体の学習計画を確認してください。

### 3. Day 1 から開始

```bash
cd Day01
open README.md  # または cat README.md
```

### 4. 毎日の流れ

1. README.md で今日のトピックを確認
2. guide.md で詳細を学習（詳細教材がある日のみ）
3. exercises.md で演習課題に取り組む
4. solution/ で模範解答を確認

## 🎓 学習のペース

### 推奨スケジュール

- **毎日 2-3時間** の学習時間を確保
- **継続が最重要**（1日休むと取り戻すのが大変）
- **週末** は復習と実践プロジェクトに充てる

### 柔軟な進め方

- ペースは自分で調整可能
- 理解できるまで同じ日を繰り返してもOK
- 余裕があれば先に進んでもOK

---

## 💡 学習のコツ

1. **手を動かす**: 読むだけでなく、必ずコードを書く
2. **理解してから進む**: わからないまま先に進まない
3. **エラーを恐れない**: エラーは最良の学習機会
4. **アウトプットする**: 学んだことをブログや SNS で発信
5. **コミュニティに参加**: React Meetup や勉強会に参加
